Жизненный цикл

Последовательность выполнения консольного приложения следующая.

script.bat --> index.php --> Bootstrap.php --> Command(s) --> Output

Пример командного файла app/Console/scripts/demo/greet.bat

php.exe ../../index.php demo:greet Petrov --yell

Важно чтобы путь к файлу php.exe был доступен.
